Sobre o Workshop

O 1º Workshop de Desenvolvimento de Sistemas foi organizado para proporcionar aos alunos do curso técnico em Desenvolvimento de Sistemas a oportunidade de apresentarem seus projetos de conclusão de curso. Venha participar deste evento e conheça 13 projetos inovadores de diferentes áreas da tecnologia, cada um com soluções criativas e inspiradoras desenvolvidas pelos novos profissionais da área de Desenvolvimento de Sistemas.

Projetos

Precision Tech


Cálculo Preciso


Danielly Cristina de Assis, Davi de Sales Lima, Diogo Pereira dos Santos, Luiz Miguel Gallo Giglioti, Henrique Linhares Pinheiro Loiola

O cálculo preciso é uma ótima solução para planejar a usinagem de peças industriais forma eficiente e sustentável. Com ele, os usuários podem especificar materiais, definir geometrias e realizar os cálculos necessários para a produção das peças. O software possibilita calcular o custo total da peça considerando diversas variáveis como: materiais, mão de obra, tempo de máquina, etc. O objetivo é auxiliar a indústria na realização de estimativas mais acertivas, consequentemente, ajudando a reduzindo os custos associados à usinagem.

FilamentsFiscal3D

Orca3D

João Paulo Trevisan, Matheus Henrique Oliveria Silva, Vinicius Gabriel Pereira Chalch e Wylgner Thomas Camargo

O Orca3D é um sistema que permite otimizar cálculos de custeio de peças em 3D. Um site foi usado para implementar esse sistema que fornece ao usuário uma ferramenta simples e útil para reportar e registrar atividades. Um dos objetivos principais é o rápido cálculo das peças e custos do equipamento.

DevLoopers


MasterNote


Erick Bessi, Flávia Gabriele Sacchi, Gustavo Peixoto de Freitas, Leonardo Aparecido Alves Nogueira, Vanessa Bernardes Teixeira

O objetivo do MasterNote é utilizar tecnologias web amplamente adotadas no mercado, para operacionalizar a aplicação da Metodologia SENAI de Ensino Profissional (MSEP). Para tanto, o intuito foi o de agilizar o processo de registro de informações para acompanhamento acadêmico das turmas pelos professores. A tarefa principal foi a de reduzir o tempo de registro manual de informações recorrentes, criar uma base de situações de aprendizagem para permitir ao docente atuar em atividades de apoio aos alunos.

DevFlow


IntelliSpace


Camila Oliveira Silva, Hiasmin Lorrane Cardoso, Kauan Gallo Gomes, Felipe Allan Da Silva, João Vitor Conceição Silva e Kaique Rodrigues Dos Santos

O IntelliSpace é um sistema de automação para o auditório do SENAI - São Carlos que busca proporcionar aos seus usuários a capacidade de controlar diferentes dispositivos tais como luzes, ar condicionado e projetor. Para isso foi utilizado tecnologia de ponta no ramo de automação residencial como como assistentes virtuais (Alexa) e dashboards interativos visando otimizar a experiência dos expectadores e palestrantes.

TechLine


MyAuto


Matheus Bruno Lorenzato, Matheus Felippe, Victor Hugo Pião Petrucelli, Adrian de Lima Pereira, Pedro Santos Cintra

O MyAuto é um aplicativo que oferece notificações inteligentes para os cuidados com o carro, como troca de óleo. Ele calcula quando deve ser realizada a próxima manutenção baseando-se na quilometragem percorrida e envia lembretes com 15 dias de antecedência.

Templários


GeoSENAI


Lucas Oliveira, Guilherme Lopes Rocetom, Gabriel Moreira da Cruz Bento, Rhuan Dhiego Rodrigues de Paulo, Davi Ponce Manieri

O GeoSENAI visa implementar um totem de autoatendimento no ambiente do SENAI para que os usuários do espaço consigam localizer os ambientes de ensino. Além disso, integra outras funcionalidades destinadas a facilitar a rotina dos estudantes.

Geeks do Mercado


Secure Lockers


Vinicius Brolezzi Gaban, Davi Vinicius Correa, Ana Rúbia Almeida, Cláudia Giovana Zacarias Francisco

O Secure Locker é um sistema de travas digitais para os armários utilizados pelos associados da AAPM com o intuito de proporcionar maior confiabilidade ao local. A ideia é oferecer segurança ao usuário ao deixar seus pertences. Para tanto, foi empregada tecnologia de última geração que oferece segurança e traz inovação para o ambiente escolar.

Unlock Safely


MEGH Safely


Mikael Ventura de Moura Braghini; Heliel Hislei Duzi Tinta; Eduardo Bruno Coppi; Gabriel Ronconi Carvalheira

O MEGH Safely é uma fechadura eletrônica que utiliza tecnologia RFID para identificar tags autorizadas e aciona uma fechadura eletrônica. O nosso projeto elimina a necessidade de chaves/códigos e permite um gerenciamento intuitivo dos armários pelos funcionários da escola garantindo a segurança e organização.

GearToolTech

GearTech

Gabriel Alves da Silva, Fabricio de Oliveira Arthur, Leonardo José Bergamin, Luigi Falconi Bruno, Marcos Vitor Matos do Santos, Thierry de Oliveira dos Santos.

O GearTech foi desenvolvido para auxiliar docentes e alunos da instituição SENAI a realizar cálculos para usinagem de peças industriais. Essa ferramenta busca diminuir a quantidade de tempo gasto na realização dos cálculos apoiando o ensino e a aprendizagem do processo de usinagem de novas peças.

MDS- Magia do Desenvolvimento de Software


CashFlow


Bruno Santiago, Camila Santana Carneiro, Gustavo Catarino Copriva, José Augusto Barbosa Ravazoli Roberto Luiz de Carvalho, Wilson Gomes da Silva Neto

O CashFlow visa desenvolver um sistema para monitorar a gestão de gastos corporativos referentes a viagens realizadas pelos colaboradores de uma empresa. O sistema é um aplicativo dedicado exclusivamente para registros de informações da viagem realizada, consequentemente, aprimorando a comunicação e eficiência durante as etapas do processo de reembolso simplificando a supervisão dos responsáveis.

Quantum Logic


Matritis


Kayke Reducino Rodrigues, Ana Médlyn Galhardo Costa, Adan Cardoso, Pablo Massao Lins Bunno, Ricardo Vanderlei Clemente Junior, Felipe de Souza Casale

O software Matritis é uma solução educacional inovadora que permite aos professores conduzir avaliações formativas e somativas de forma rápida e confiável. Com recursos avançados de análise e relatórios detalhados, o Matritis oferece uma visão abrangente do progresso dos alunos que facilita a identificação de áreas de melhoria, ou seja, auxiliam na tomada de decisão durante o processo de ensino.

Code in Air


Code in Air


Pedro Emilio Scarlato Basso, Victor Hugo Pinesso Fusca, Victor Gabriel Farias Da Rocha, Rhian Henrique Custódio, André Guimarães, Rafael Eduardo Soares Dos Santos

O Code in Air é voltado para estudantes e professores de pneumática e hidráulica tem como objetivo facilitar os estudos e armazenar cálculos mais precisos para otimização do tempo.

TechBuilders


TechLife


Miguel Lima Pereira Da Cruz, Miguel Henrique, Luís Gustavo, Hugo Correia Luiz, Leonardo Amaral

O TechLife utiliza sensores inteligentes e Sonoff para otimizar a experiência do uso de ambientes de ensino. Este projeto de automação por meio de um aplicativo visa criar ambientes mais sustentáveis e também mais confortaveis para seus utilizadores.
